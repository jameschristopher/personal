Alphalens - a tool for anayzing stock alpha factors

Alphalens is a Python package designed to evaluate equity return predicting factors. These factors are commonly used in alpha factor modeling. The main function of Alphalens is to surface the most relevant statistics and plots about an alpha factor, including:

  • Returns Analysis
  • Information Coefficient Analysis
  • Turnover Analysis
  • Sector Analysis

Alphalens is ideally ran inside a Jupyter notebook with just a few lines of code:

import alphalens

alphalens.tears.create_factor_tear_sheet(my_factor, pricing)
